{% highlight bash %}
vim ~/.ssh/config
{% endhighlight %}
贴入如下内容:
{% highlight bash %}
host *
ControlMaster auto
ControlPath ~/.ssh/master-%r@%h:%p
{% endhighlight %}
退出iterm2，重新登录即可。
