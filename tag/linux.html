让VIM记录文件上次编辑的位置

2016-03-11 Beijing

在~/.vimrc中加入如下配置:

autocmd BufReadPost *
\ if line("'\"")>0&&line("'\"")<=line("$") |
\ exe "normal g'\"" |
\ endif
autocmd BufReadPost *
\ if line("'\"")>0&&line("'\"")<=line("$") |
\ exe "normal g'\"" |
\ endif
syntax enable
set nu
