让VIM记录文件上次编辑的位置

2016-03-11 Beijing

在~/.vimrc中加入如下配置:

autocmd BufReadPost *
\ if line("'\"")>0&&line("'\"")<=line("$") |
\ exe "normal g'\"" |
\ endif
autocmd BufReadPost *
\ if line("'\"")>0&&line("'\"")<=line("$") |
\ exe "normal g'\"" |
\ endif
syntax enable
set nu

Iterm2 Clone Session

2015-05-20 Beijing 
vim ~/.ssh/config

贴入如下内容:

Host *
    ServerAliveInterval 30
    ControlMaster auto
    ControlPath ~/.ssh/master-%r@%h:%p
    ControlPersist yes

退出iterm2，重新登录即可。
